How to Use Clone Layer
1. Feed clean operating data

Use this hub daily to push priorities, meeting outcomes, blockers, and style preferences. Better input quality creates better clone output quality.

Rule: facts first, opinions second
2. Run Talk Mode as command center

Open Talk Mode and issue direct instructions: “Prioritize my day”, “Draft high-value replies”, “Show what needs approval first”.

Use short direct commands
3. Let clone learn your style

Say explicit lines like “I prefer concise updates” or “My top priority is admissions growth.” Clone stores these and adapts over time.

Review in learned-memory panel
4. Keep execution gated

Use clone for drafting and planning. Risky actions should stay approval-gated. This keeps speed high without losing control.
