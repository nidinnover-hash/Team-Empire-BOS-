How to Use Agent Layer
1. Feed clean operating data

Use this hub daily to push priorities, meeting outcomes, blockers, and style preferences. Better input quality creates better agent output quality.

Rule: facts first, opinions second
2. Use Agent Chat to run the AI Agent Factory

Open Agent Chat and issue direct instructions: "Prioritize my day", "Draft high-value replies", "Show what needs approval first".

Use short direct commands
3. Let the agent learn your style

Say explicit lines like “I prefer concise updates” or “My top priority is admissions growth.” The agent stores these and adapts over time.

Review in learned-memory panel
4. Keep execution gated

Use the agent for drafting and planning. Risky actions should stay approval-gated. This keeps speed high without losing control.
