# Go-To-Market: Pricing + Legal Positioning

## Positioning

Nidin BOS is a `suggest-only` executive operating system.
It centralizes visibility, compliance monitoring, and decision support while
keeping irreversible actions behind human approval.

Core promise:
- Faster CEO decisions
- Better compliance hygiene
- No autonomous destructive actions

## Packaging

### Starter
- Single organization
- Core integrations (GitHub, ClickUp, Slack, DigitalOcean)
- Daily CEO summary
- Basic compliance report

### Growth
- Multi-organization support
- Compliance policies + violation workflow
- Scheduler job telemetry and replay
- Integration SLA monitoring

### Enterprise
- Advanced access governance
- Contractual SLA and support escalation
- Security/compliance review pack
- Audit export + data retention controls

## Usage Metering

Recommended blended model:
- Base platform fee (tier-based)
- Seat fee for active users
- Usage add-ons for high-volume sync and AI token usage

## Legal Positioning

Required legal package:
- Master Services Agreement (MSA)
- Data Processing Addendum (DPA)
- Privacy Policy
- Subprocessor list

Critical clauses:
- Suggest-only enforcement boundary
- No irreversible action without explicit user confirmation
- Customer-controlled retention and deletion windows
- Security incident notification timeline

## Security Claims You Can Defend

- Tenant scoping by `organization_id` across operational tables
- Token encryption at rest
- Structured audit events with actor and request traceability
- Role-based access controls and approval gates for sensitive actions
